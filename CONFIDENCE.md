## 0. 票代原则（Ticket Agent Principle）

> 你让AI帮你买机票，它会搜遍全网给你一个携程或飞猪的"最低价"，然后信心十足地告诉你这就是最优解。
> 它不会告诉你还有一层价格体系的存在，因为它根本不知道。
> ——《躺平指数》

**推论**：AI给你的分析**永远是公开信息的最优解**，不是真实世界的最优解。两者之间的价差，来自AI无法触及的信息层。

## 1. 信息四层级体系

| 层级 | 定义 | Alpha潜力 | 数据来源 | AI可获取 | 示例 |
|------|------|----------|---------|---------|------|
| **L0** | 广泛公开信息 | 0（已定价） | 财报/K线/新闻 | ✅ | PE=25、MACD金叉、营收数据 |
| **L1** | 公开但需解读 | 有限 | 公开数据+领域框架 | ✅需框架 | 行业对比、DCF估值、护城河分析 |
| **L2** | 半公开/从业者知识 | 潜在 | 行业会议/供应链/从业者 | ⚠️碎片 | 渠道库存体感、订单趋势、政策预期 |
| **L3** | 非公开/经验直觉 | 高 | 个人经验/人脉/实地调研 | ❌不可及 | 管理层语气变化、茶歇判断、消费趋势体感 |

**关键认知**：
- L0+L1 = **共识层**：AI擅长，但无超额收益
- L2+L3 = **非共识层**：Alpha来源，AI无法系统获取
- **真正的投资洞见几乎总是位于L2-L3层**

## 2. 双重盲区警告

### 2.1 基本面盲区

```
基本面报告给你的感觉：地图上少了几条路，你看得到空白
实际情况：AI不知道管理层在想什么、供应链真实体感如何、
         行业会议上私下流传的判断、消费趋势的末端变化
危险：这份"完整"的报告让你误以为已经了解了公司
```

### 2.2 技术面盲区

```
技术分析给你的感觉：数据完备，指标精确，无懈可击
实际情况：同样的头肩顶在牛市末期=反转，在强势回调=洗盘
         K线图一模一样，含义完全相反
         错误不藏在数据里，藏在"解读"里
危险：完备≠正确。数据没有空白让你失去了警惕
```

### 2.3 交叉验证错觉

```
基本面报告缺了你看不见的信息层
技术面报告缺了你看不见的市场语境
两份报告各自盲区你都没察觉
叠在一起却产生"互相印证"的错觉
→ 信念感更强、仓位更重、止损更慢
```

## 3. 信息效率悖论

> 华尔街几十年前就想明白了：所有你能从公开渠道拿到的信息，已经反映在股价里了。
> 半强有效市场：公开信息不产生超额收益。
> AI让你处理公开信息的速度从三天变三分钟——但信息没有因此产生更大价值。

**结论**：AI的效率提升是真实的，但效率提升的对象恰恰是**最不值钱的共识信息**。

## 4. 「知道自己不知道」检查清单

每份分析输出前，必须完成以下自检：

### A. 信息层级声明
```
本分析基于的信息层级：
  L0数据：[具体列出]  ← 已定价，无Alpha
  L1解读：[具体列出]  ← 需要独特框架才有Alpha
  L2半公开：[列出任何可获取的]  ← 如有，标注获取渠道
  L3缺失：[列出AI确定无法获取但重要的信息]  ← 强制填写
```

### B. 盲区声明
```
本分析可能遗漏的关键信息（AI无法获取）：
  1. [行业/公司]的真实[维度]体感：无法从公开数据获取
  2. 管理层在[场合]的非公开表态：不在任何数据库中
  3. [供应链/渠道]的前瞻信号：仅从业者可感知
```

### C. 置信度校准
```
基于信息层级，本报告的置信度上限：
  L0主导 → 置信度≤50%（纯共识，不产生Alpha）
  L1主导 → 置信度≤65%（框架解读有价值，但仍属公开信息）
  L2部分 → 置信度≤75%（有半公开信源，但需交叉验证）
  无L3信息 → 置信度不可超过80%（任何分析都如此）
```

### D. 旧鸿沟vs新鸿沟
```
旧鸿沟（已抹平）：散户缺工具→AI补上了
新鸿沟（警惕）：理解AI输出边界的人 vs 不理解的人
→ 这份报告可能增加你的conviction（信念感），但这不代表判断更正确
```

## 5. 自我检查问题（每份分析前必问）

1. **"这份分析里，哪部分是任何人都能用公开数据得出的共识结论？"**
2. **"如果我是这个行业的从业者，我会觉得这份分析缺了什么？"**
3. **"有哪些信息，AI根本不知道它自己不知道？"**
4. **"这份报告的'完整性'会不会让我产生虚假的安全感？"**
5. **"上个月类似的共识分析，准确率是多少？（查MEMORY）"**

## 6. 投资中的「地图与封路」比喻

```
基本面盲区 = 地图上少了几条路 → 你看得到空白，心里有数
技术面盲区 = 地图上路都画全了，但没人告诉你哪条路今天封了 → 你满怀信心出发，撞上路障才知道走错了

AI给你的是一张前所未有的详尽地图。
但走哪条路、什么时候出发，地图不会替你决定。
```
