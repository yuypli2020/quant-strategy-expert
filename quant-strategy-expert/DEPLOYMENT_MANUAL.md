## 1. 系统概览

### 1.1 设计原则

- **模块化 + 阶段推进**：每次只做简单、可验证的任务
- **新闻信号触发**：黄金坑、板块早信号、宏观事件
- **任务追踪**：维护任务清单，自动更新状态
- **宪法优先**：融合派优先级不可违背

### 1.2 六阶段执行路线图

| 阶段 | 名称 | 触发 | 输出 |
|------|------|------|------|
| 阶段0 | 初始化 | 首次运行 | 任务清单 + 蓝图 |
| 阶段1 | 宏观扫描 | 每周/事件 | 宏观报告 + 板块机会 |
| 阶段2 | 深度验证 | 指定标的 | 四框架评分 |
| 阶段3 | 信号融合 | 阶段2后 | 定罪分数 |
| 阶段4 | 执行协议 | 阶段3后 | 仓位 + 止损 |
| 阶段5 | 风险堡垒 | 执行前 | 风险评级 |
| 阶段6 | 进化复盘 | 交易后 | 规则更新 |

## 3. 融合派框架

### 3.1 四框架

| 框架 | 用途 | 权重 |
|------|------|------|
| 缠论周线 | 判断方向（核心） | 30% |
| 波浪理论 | 定位浪型 | 20% |
| 板块轮动 | 资金验证 | 15% |
| 事件驱动 | 催化时机 | 15% |
| 新闻信号 | 情绪辅助 | 20% |

### 3.2 共振评分算法

```javascript
综合评分 = 
  (缠论分 × 0.30) +
  (波浪分 × 0.20) +
  (板块分 × 0.15) +
  (事件分 × 0.15) +
  (新闻信号分 × 0.20)
```

### 3.3 操作建议

| 综合评分 | 操作 | 仓位 |
|----------|------|------|
| 4.5~5.0 | 完美共振，重仓 | 80-100% |
| 4.0~4.4 | 强共振，正常持仓 | 60-80% |
| 3.0~3.9 | 弱共振，轻仓试探 | 30-50% |
| 2.0~2.9 | 无共振，观望 | 10-20% |
| <2.0 | 负共振，清仓 | 0% |

### 3.4 宪法级优先级

> **缠论周线是核心，其他框架不能违背缠论信号。**

- 缠论周线看空 → 即使其他框架再好也不做多
- 缠论周线看多 → 其他框架辅助寻找更好买点

## 4. 新闻信号触发

### 4.1 三类核心信号

| 信号 | 定义 | 优先级 |
|------|------|--------|
| 黄金坑 | 横盘后中大阴线挖坑，快速反弹确认 | ★★★★★ |
| 板块早信号 | 政策催化/供应链消息/机构动向 | ★★★★ |
| 宏观事件 | 降息/加息/非农/CPI | ★★★★ |

### 4.2 黄金坑判定

```
条件：
1. 3日内跌幅 8-15%
2. 此后横盘至少 20 交易日
3. 止跌确认（长下影线/锤子线/阳线反包）
```

### 4.3 板块早信号判定

```
催化类型：
- 顶层政策（国务院）→ 5分
- 部委政策 → 4分
- 行业会议/协会 → 3分
- 供应链消息 → 3分
- 机构动向 → 2分
```

## 5. 执行协议

### 5.1 ATR止损

```javascript
止损位 = 入场价 - ATR(14) × 2
```

### 5.2 3R仓位计算

```javascript
风险金额 = 账户 × 2%
仓位 = 风险金额 / 止损幅度
```

### 5.3 风险堡垒检查

| 风险等级 | 条件 | 操作 |
|----------|------|------|
| 🟢 安全 | 所有检查通过 | 正常交易 |
| 🟡 警戒 | 1-2项超标 | 降低仓位50% |
| 🔴 危险 | 3项以上超标 | 停止开新仓 |
| ⚫ 熔断 | 单日亏损>5% | 清仓观望 |

## 7. 快速启动

### 7.1 阶段0：初始化

```
用户：「初始化 Genesis」
Agent：
1. 创建任务追踪文件
2. 加载所有 skill
3. 输出完整蓝图
4. 询问进入哪一阶段
```

### 7.2 阶段1：宏观扫描

```
用户：「扫描市场」
Agent：
1. 获取指数行情
2. 获取板块排行
3. 获取新闻资讯
4. 分析宏观事件
5. 输出：宏观报告 + 板块机会清单
```

### 7.3 阶段2：深度验证

```
用户：「分析 [标的]」
Agent：
1. 缠论周线分析
2. 波浪理论定位
3. 板块轮动验证
4. 事件驱动分析
5. 新闻信号分析
6. 计算融合派评分
7. 输出：四框架评分 + self_refutation
```
