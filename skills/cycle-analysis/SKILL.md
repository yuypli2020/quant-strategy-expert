# cycle-analysis — 周期分析流派 V28

> **版本**：V28  
> **来源**：合并 `chan-theory`（缠论）+ `elliott-wave`（波浪）  
> **核心理念**：缠论找精确买卖点，波浪看结构位置，两者互补。周期是市场最重要的维度之一，但不是唯一。


## Part A：缠论买卖点判定

### A1. 周线MA20（宪法级均线）
```
周线MA20 = sum(最近20周收盘价) / 20
当前周收盘 > 周线MA20 → 多头信号
当前周收盘 < 周线MA20 → 空头信号
差距 < 3% → 边界信号（模糊区）
```

### A2. MACD方向验证
```
MACD柱 > 0 且 DIF > DEA → 多头
MACD柱 < 0 且 DIF < DEA → 空头
其他 → 震荡
```

### A3. 买卖点判定（基于缠论简化规则）

```
买点可能：
  一买：从近期高点回落≥25%，出现第一根周阳线 → 尝试买入
  二买：一买后回调不破前低，周线再次转阳 → 确认买入
  三买：上升中继后突破新高 → 追涨买入（高风险）

卖点可能：
  一卖：从近期低点反弹≥25%，出现第一根周阴线 → 止盈/止损
  二卖：一卖后反弹不破前高，周线再次转阴 → 确认卖出
  三卖：下降中继后跌破新低 → 杀跌卖出（极端情况）
```

### A4. 缠论综合评分（50%权重）
```
缠论分 = MA20信号×0.4 + MACD信号×0.3 + 买卖点×0.3

MA20多头 + MACD多头 + 买点信号 → 5分 ★★★★★
MA20多头 + MACD多头 + 无买卖点信号 → 4分 ★★★★
MA20多头 + 震荡MACD → 3分 ★★★
MA20边界 + 震荡 → 2分 ★★
MA20空头 + MACD空头 → 1分 ★
```

## Part B：波浪理论判定

### B1. 识别关键高低点
```
从500根日线中识别：
  - 近1年最高点和日期
  - 近1年最低点和日期
  - 近期次高点/次低点

摆动定义：相邻高低点差≥15%才算有效摆动
```

### B2. 斐波那契回撤分析
```
从高点到低点的回撤比例：
  回撤 ≤ 38.2% → 可能是强势2浪（看多）
  回撤 38.2-61.8% → 标准2浪（看多）
  回撤 61.8-78.6% → 可接受2浪（谨慎看多）
  回撤 > 78.6% → 可能不是2浪，是ABC调整（中性偏空）
```

### B3. 当前浪型判定

| 浪型 | 特征 | 信号 |
|------|------|------|
| **3浪** | 从回调低点上涨，幅度 > 1浪×1.618 | 🔴 强烈看多 |
| **5浪** | 连续第5波上涨，幅度可观 | 🟡 看多但注意回调 |
| **1浪** | 从低点刚启动，幅度有限 | 🟡 确认后再买 |
| **A浪/B浪** | 下跌后反弹，不能确认底部 | ⚠️ 观望 |
| **C浪** | 连续下跌，创出新低 | 🔴 看空 |
| **2浪/4浪** | 回调中，不破前低 | 🟢 等待止跌信号 |

### B4. 波浪综合评分（50%权重）
```
波浪分 = 浪型信号×0.6 + 斐波比例验证×0.4

3浪 + 斐波验证 → 5分 ★★★★★
5浪 + 斐波验证 → 4分 ★★★★（但要减仓）
1浪启动 → 3分 ★★★（等确认）
2浪/4浪回调中 → 2分 ★★（等止跌）
C浪确认 → 1分 ★（清仓）
```

## Part C：综合评分

```
周期分析总分 = 缠论分×0.5 + 波浪分×0.5

| 总分 | 周期判断 | 建议 |
|------|---------|------|
| 4.5-5.0 | 上升周期3/5浪中段 | 持有或加仓 |
| 3.5-4.4 | 周期向上但位置偏高 | 持有，注意止盈 |
| 2.5-3.4 | 震荡周期，中性 | 观望或区间操作 |
| 1.5-2.4 | 周期向下2/A浪 | 减仓，等一买/二买 |
| <1.5 | 明确下降C浪 | 清仓或做空 |
```
