## 三、阶段执行流程

### 阶段0：初始化

```
目标：加载上下文，建立追踪系统
输出：完整蓝图 + 任务清单
下一步：进入阶段1或指定股票
```

### 阶段1：宏观扫描

```
目标：PMI/CPI矩阵 + 板块RS + 新闻信号
触发：每周一 或 重大事件
输出：宏观报告 + 板块机会清单
下一步：进入阶段2（选定的股票/板块）
```

### 阶段2：深度交叉验证

```
目标：个股/板块完整分析
输入：选定的标的
输出：四框架评分 + 信号摘要
下一步：进入阶段3
```

### 阶段3：信号融合 + 定罪分数

```
目标：融合所有信号，计算最终评分
输出：定罪分数 + 操作建议
下一步：进入阶段4
```

### 阶段4：执行协议

```
目标：仓位计算 + ATR止损
输出：执行清单 + 风险检查
下一步：进入阶段5
```

### 阶段5：风险堡垒

```
目标：最终风险检查
输出：风险评级 + 通过/不通过
下一步：执行交易 或 进入阶段6
```

### 阶段6：进化复盘

```
目标：验证信号有效性，提炼规则
输出：更新规则库 + 新的任务
触发：交易后 或 每周
```
