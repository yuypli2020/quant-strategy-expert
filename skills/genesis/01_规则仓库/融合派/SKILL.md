# 融合派 SKILL.md

## 策略概述

融合派 = 四框架共振 × 缠论优先

**核心逻辑**：
1. 缠论周线判断方向（核心，决定能不能做）
2. 波浪理论定位浪型（辅助，判断在哪个阶段）
3. 板块轮动验证资金（过滤，决定做哪个）
4. 事件驱动寻找催化（时机，决定什么时候做）

**优先级**：缠论周线 > 波浪 > 板块 > 事件
**不能违背**：缠论看空则不做，缠论看多才考虑

---

## 一、缠论周线判定规则

### 1.1 笔的定义

```
上升笔 = 底分型 + n根阳线 + 顶分型（n ≥ 5）
下跌笔 = 顶分型 + n根阴线 + 底分型（n ≥ 5）
```

**简化判定**（用K线形态代替分型）：
- 顶分型：中间K线高点最高，低点也最高
- 底分型：中间K线低点最低，高点也最低

### 1.2 买卖点判定

| 信号 | 条件 | 评分 |
|------|------|------|
| **一买** | 下跌笔后出现底分型，且突破前一下跌笔高点 | ★★★★ |
| **二买** | 一买后回调不破一买低点，出现底分型 | ★★★★★ |
| **类二买** | 突破前中枢高点后回调不破中枢上沿 | ★★★★ |
| **三买** | 回调不破第二个中枢上沿 | ★★★★ |
| **一卖** | 上升笔后出现顶分型，且跌破前一上升笔低点 | ★★★★ |
| **二卖** | 一卖后反弹不破一卖高点，出现顶分型 | ★★★★ |

### 1.3 简化MA20判定（用于回测）

```
周线MA20 = 最近20周收盘价的简单平均

买入信号：金叉（上周收盘≤MA20 AND 本周收盘>MA20）
卖出信号：死叉（上周收盘≥MA20 AND 本周收盘<MA20）
```

### 1.4 缠论评级

| 评级 | 缠论状态 | 操作 |
|------|----------|------|
| ★★★★★ | 二买/三买 | 重仓介入 |
| ★★★★ | 一买/类二买 | 正常买入 |
| ★★★ | 上升中继（回调不破前低） | 持有 |
| ★★ | 震荡/中性 | 轻仓/观望 |
| ★ | 一卖/二卖 | 清仓/不做 |

## 二、波浪理论判定规则

### 2.1 浪型识别

**1浪**：从底部快速拉升，突破前高
- 特征：成交量放大，斜率陡峭
- 判定：缠论一买后，突破前期高点

**2浪**：回撤1浪
- 特征：回撤1浪的38.2%~78.6%
- 判定：缠论回调笔，不破1浪起点

**3浪**：主升浪
- 特征：最强劲，持续时间长，幅度大
- 判定：缠论上升笔延续，创阶段新高

**4浪**：回撤3浪
- 特征：回撤3浪的38.2%以内
- 判定：缠论回调笔，不破3浪的38.2%

**5浪**：末端
- 特征：上升动能减弱，可能背离
- 判定：缠论上升笔但创新高力度减弱

### 2.2 波浪评级

| 评级 | 浪型 | 说明 | 操作 |
|------|------|------|------|
| ★★★★★ | 3浪 | 主升浪 | 重仓持有 |
| ★★★★ | 5浪起点 | 末端上涨 | 分批减仓 |
| ★★★ | 1浪/5浪中段 | 趋势中 | 正常持仓 |
| ★★ | 2浪/4浪 | 调整浪 | 观望/轻仓 |
| ★ | C浪 | 下跌浪 | 清仓 |

### 2.3 斐波那契比例

```
2浪回撤 = 1浪 × 0.382 ~ 0.618（强势0.382，弱势0.618）
4浪回撤 = 3浪 × 0.382（通常浅回撤）
3浪长度 = 1浪 × 1.618~2.618
```

## 三、板块轮动规则

### 3.1 板块评级算法

```
板块评分 = 5日涨幅分 × 0.3 + 20日涨幅分 × 0.3 + 60日涨幅分 × 0.2 + 量比分 × 0.2
```

**分计算**：
- 排名前10% → 5分
- 排名10%~30% → 4分
- 排名30%~50% → 3分
- 排名50%~70% → 2分
- 排名70%以后 → 1分

### 3.2 板块选股原则

| 原则 | 说明 |
|------|------|
| 只做强势板块 | 缠论看多的标的必须在强势板块 |
| 回避弱势板块 | 即使技术形态好，板块弱势也不做 |
| 等板块企稳 | 板块指数站稳MA20再考虑个股 |

### 3.3 板块评级

| 评级 | 板块状态 |
|------|----------|
| ★★★★★ | 龙头板块，5日涨幅前5%，资金持续流入 |
| ★★★★ | 强势板块，5日涨幅前20% |
| ★★★ | 轮动板块，涨幅中等 |
| ★★ | 弱势板块，5日涨幅后50% |
| ★ | 资金流出，趋势向下 |

## 四、事件驱动规则

### 4.1 事件分类

| 分类 | 类型 | 确定性 | 操作建议 |
|------|------|--------|----------|
| A类 | 重大政策/业绩超预期 | 高 | 可追涨 |
| B类 | 行业政策/产品发布 | 中 | 轻仓试探 |
| C类 | 无重大事件 | - | 纯技术 |
| D类 | 重大利空 | 高 | 回避 |

### 4.2 事件评分

| 评级 | 事件类型 |
|------|----------|
| ★★★★★ | 重大政策加码（如降准、AI补贴） |
| ★★★★ | 业绩超预期20%以上 |
| ★★★ | 产品发布/行业大会 |
| ★★ | 无重大事件（纯技术机会） |
| ★ | 重大利空/黑天鹅 |

### 4.3 事件与缠论结合

```
缠论买点 + 事件催化 → 加仓
缠论买点 + 无事件 → 正常仓位
缠论买点 + 利空事件 → 观望
缠论卖点 + 利好事件 → 减仓但不全清
缠论卖点 + 无事件 → 正常减仓
```

## 五、融合派共振算法

### 5.1 评分矩阵

| 框架 | ★★★★★ | ★★★★ | ★★★ | ★★ | ★ |
|------|--------|-------|------|-----|---|
| 缠论 | 二买/三买 | 一买/类二买 | 上升中继 | 震荡 | 一卖/二卖 |
| 波浪 | 3浪 | 5浪起点 | 1浪/5浪中段 | 2/4浪 | C浪 |
| 板块 | 龙头+ | 强势 | 轮动 | 弱势 | 资金流出 |
| 事件 | 重大利好 | 业绩超预期 | 产品催化 | 无事件 | 重大利空 |

### 5.2 综合评分

```
综合评分 = (缠论分 × 0.4) + (波浪分 × 0.25) + (板块分 × 0.2) + (事件分 × 0.15)
```

### 5.3 操作建议

| 综合评分 | 操作 | 仓位 |
|----------|------|------|
| 4.5~5.0 | 完美共振，重仓 | 80-100% |
| 4.0~4.4 | 强共振，正常持仓 | 60-80% |
| 3.0~3.9 | 弱共振，轻仓试探 | 30-50% |
| 2.0~2.9 | 无共振，观望 | 10-20% |
| <2.0 | 负共振，清仓 | 0% |

## 六、执行模板

### 6.1 分析流程

```
用户：「分析 XXX」

1. 获取标的代码和名称
2. 获取周线数据（westock-data kline --period week）
3. 获取技术指标（westock-data technical --group ma,macd,rsi）
4. 获取板块信息（westock-data sector）
5. 获取近期新闻（westock-data news）

然后按以下顺序计算：

Step 1: 缠论评级
  - 计算MA20
  - 判断金叉/死叉
  - 识别买卖点
  → 缠论分

Step 2: 波浪评级
  - 分析浪型结构
  - 判断当前所处阶段
  → 波浪分

Step 3: 板块评级
  - 获取板块5日/20日/60日涨幅
  - 计算板块评分
  → 板块分

Step 4: 事件评级
  - 分析近期新闻和公告
  - 判断事件类型
  → 事件分

Step 5: 融合派评分
  - 综合评分 = Σ(分×权重)
  - 确定操作建议

Step 6: 输出报告
```

## 七、Self-Refutation 模板

```
【Self-Refutation】

可能错误的原因：
1. 周线滞后：金叉信号确认时可能已上涨10%+
2. 震荡市失效：频繁金叉死叉导致止损

失效边界：
- 跌破周线MA20 → 止损
- 板块轮动转弱 → 减仓
- 重大利空 → 清仓

数据局限性：
- 周线每周只更新5个数据点
- 无法识别日内大幅波动
```
