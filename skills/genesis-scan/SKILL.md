| # | 模式 | 判断什么 | 数据来源 | 权重 |
|---|------|---------|---------|------|
| **①黄金坑** | 超跌反弹机会 | kline + technical + news | 高（高赔率） |
| **②板块早信号** | 行情启动初期 | sector --rank + marketnews | 高（先发优势） |

| **⑤情绪温度** | 整体市场热度 | quote + strategy涨停跌停 | 辅助 |

## Step ①：黄金坑扫描（最重要，高赔率机会）

**原理**：大跌后横盘，不是止损位，是黄金坑。

```bash
# 全市场跌幅榜（找跌幅>8%的标的）
westock-tool filter "ChangePCT < -8" --limit 20

# 对跌幅>8%的标的，拉K线验证
westock-data kline <code> --period day --limit 40
westock-data technical <code> --group boll,rsi

# 查相关新闻（为什么跌？）
westock-data news <code> --limit 10 --type 2
```

### 黄金坑判定规则

```
黄金坑 = 跌幅条件 AND 周期条件 AND 止跌确认

跌幅条件：3日内跌幅 8-15%
周期条件：此前横盘至少 20 个交易日（价格在±15%内震荡）
止跌确认：满足以下任一
  → 长下影线（下影线 > 实体 2倍）
  → 锤子线（收盘 > 开盘，且下影线 > 实体）
  → 次日阳线反包
```

### 评分公式

```
黄金坑分 = 跌幅分×0.3 + 止跌分×0.3 + 新闻情绪分×0.4

跌幅分：
  8-10% → 4分  ← 黄金区间
  10-12% → 5分
  12-15% → 4分（跌太多可能不是坑）
  <8% 或 >15% → 0分（不是黄金坑）

止跌分：
  长下影线/锤子线 → 5分
  次日阳线反包 → 4分
  其他止跌形态 → 3分

新闻情绪分（看marketnews和news）：
  恐慌蔓延 → -2分（坏）
  澄清公告 / 靴子落地 → +3分（好）
  无新闻 → +1分（中性）
```

### 黄金坑信号等级

| 分值 | 等级 | 建议 |
|------|------|------|
| 4.0-5.0 | 🟢🟢 强黄金坑 | 重点关注，可进观察池 |
| 3.0-3.9 | 🟢 黄金坑 | 关注，等止跌确认 |
| 2.0-2.9 | ⚠️ 疑似 | 观望，不急着进 |
| <2.0 | ❌ 不是黄金坑 | 跳过 |

## Step ②：板块早信号扫描

**原理**：板块行情刚启动时介入，不是追高。

```bash
# 申万一级行业（看谁刚启动）
westock-data sector --rank interval_chg_rank_sw1 --sort chg5Days --limit 20

# 聚源概念（找主题催化）
westock-data sector --rank interval_chg_rank_industry --sort chg5Days --limit 30

# 查催化原因
westock-data marketnews hs
westock-data calendar --limit 10 --country 1 --indicator 2
```

### 板块早信号判定

```
板块早信号 = 催化类型分×0.4 + 时机分×0.3 + 强度分×0.3

催化类型分：
  顶层政策（国务院/中央） → 5分
  部委政策（工信部/，发改委） → 4分
  行业会议/协会文件 → 3分
  供应链消息（涨价/限售） → 3分
  机构动向（大股东增持） → 2分
  无明显催化 → 1分

时机分：
  5日涨幅 3-8%（刚启动） → 5分
  5日涨幅 8-15%（中段） → 3分
  5日涨幅 >15%（末端） → 1分

强度分：
  多渠道确认（新闻+公告+资金） → 5分
  双渠道确认 → 3分
  单渠道 → 2分
```

### 信号等级

| 分值 | 等级 | 建议 |
|------|------|------|
| 4.0-5.0 | 🔥 早信号确认 | 可进观察池 |
| 3.0-3.9 | 🟡 有催化 | 关注催化持续性 |
| 2.0-2.9 | ⚠️ 弱信号 | 观察 |
| <2.0 | ❌ 无信号 | 跳过 |

## Step ④：M因子快检（唯一仓位否决）

```bash
westock-data kline sh000300 --period day --limit 60
```

| M评分 | 大盘 | 仓位上限 |
|-------|------|---------|
| 8-10 | 强势 | 100% |
| 6-7 | 偏强 | 80% |
| 4-5 | 中性 | 60% |
| 2-3 | 偏弱 | **30%** ⚠️ |
| 0-1 | 弱势 | **10%** 🔴 |

**M < 4 时，报告顶部强制标注仓位上限。**

## Step ⑤：情绪温度

```bash
westock-data quote sh000001 sz399001 sz399006
westock-tool strategy daily_limit_up --limit 10
westock-data hot board --limit 10
```

情绪温度判定：
- 涨停 >50 + M因子强势 → 贪婪（谨慎追高）
- 跌停 >30 + M因子弱势 → 恐惧（关注黄金坑）
- 涨跌家数比正常 → 中性

## 观察池汇总 + 优先级排序

```
排序规则：
  ①黄金坑分 × 0.35  ← 最重要，高赔率
  ②板块早信号分 × 0.25
  ③材料短缺信号分 × 0.20
  ④M因子仓位上限归一化 × 0.10
  ⑤情绪温度 × 0.10

最终分 = Σ(各分×权重)
```

## Self-Refutation（必须3点）

1. **黄金坑可能变真坑**：跌15%以上可能是基本面恶化，不是机会。需要验证新闻是"靴子落地"还是"业绩暴雷"。
2. **板块早信号可能是假信号**：政策预期未兑现就会跌回。需要多渠道验证催化真实性。
3. **材料短缺有滞后**：板块涨幅已反映过去5日，新闻可能提前定价。需要对比marketnews时间戳。
