## 执行流程

### Step 1: 数据拉取（全部7条，不可省略）

```bash
# 技术面（日线）
westock-data kline <代码> --period day --limit 260
westock-data technical <代码> --group boll,rsi,macd

# 🔴 CL-W必须（周线 + 月线）
westock-data kline <代码> --period week --limit 80
westock-data kline <代码> --period month --limit 24

# 基本面 + 资金面
westock-data finance <代码> --num 12
westock-data asfund <代码>

# 情绪面
westock-data lhb <代码>            ← 龙虎榜(DRGN)
westock-data news <代码> --limit 15
westock-data hot                   ← 热搜(SENT/DRGN)
```

### Step 2: 七学派逐一输出信号

每个学派输出：
- 信号方向（🟢买入/🔴卖出/🟡观望）
- 信号强度（0-10分）
- 核心理由（1-2句）
- 数据证据（具体指标值）

### Step 3: 信号合成

```
合成得分 = Σ(学派信号强度 × 学派权重) × 共振加成系数 × 风控校验

共振加成：
  2派同向 ×1.2
  3派同向 ×1.5
  4派同向 ×1.8
  5派以上 ×2.0

风控校验：
  宪法IR规则否决 → 得分归零
  M因子<4 → 最高仓位30%
```

### Step 4: CL-W周线裁判裁定

```
CL-W裁判权：
  周线上升趋势确认 → ✅可否决卖出信号
  周线下降趋势确认 → ✅可否决买入信号
  周线中枢震荡 → ❌不否决，退出裁判
  日线级别结构 → ❌日线不具备裁判资格
```

### Step 5: 冲突辩论触发

当3个以上学派信号方向相反时，触发辩论：
```
辩论格式：
  「正方」TF + MOM + DRGN → 买入理由
  「反方」MR + EMO → 卖出/观望理由
  「裁判」CL-W → 周线结构裁定
  「结果」最终方向 + 信心度
```

### Step 6: 输出诊断报告

## 🔴 CL-W 周线缠论分析流程（V26 核心）

### 数据要求

```
必须：周线K线（--period week --limit 80）
必须：月线K线（--period month --limit 24）
辅助：日线K线（--period day --limit 260）

不可仅用日线做缠论分析。
```

### 周线分型判定

**底分型**：连续三根周K线，中间最低价 < 两侧最低价，右侧收盘 > 中间收盘
**顶分型**：连续三根周K线，中间最高价 > 两侧最高价，右侧收盘 < 中间收盘

### 周线笔/中枢

- 周线笔：始于底分型终于顶分型（或反之），长度≥5根周K线
- 周线中枢：至少3段重叠，规模≥6个月
- 中枢突破：价格有效突破中枢上下沿 + 回踩确认

### 三重买卖确认

买入：
1. **结构完成**：周线底分型确认
2. **动能验证**：周线笔量能放大 + 实体增长
3. **级别协同**：月线支撑 + 日线同向

卖出：
1. **结构完成**：周线顶分型确认
2. **动能验证**：周线向下笔量能激增
3. **级别协同**：月线压力 + 日线同向

## DRGN 游资龙头识别（V26 新增）

### 龙头四维判定

| 维度 | 标准 | 权重 |
|------|------|------|
| 板块涨幅 | 标的所在板块当日涨幅TOP3 | 30% |
| 成交额 | 板块内成交额第一 | 25% |
| 龙虎榜 | 机构净买入板块内最大 | 25% |
| 热搜 | 热搜榜排名靠前 | 20% |

**确认条件**：综合得分≥7.0为龙头，≥5.0为龙二，<5.0为跟风（不做）

### 分歧转一致节点（信号强度0.9）

```
分歧信号 → 一致信号 → 拐点确认
  板块涨跌分化大  → 跟风补涨 → 龙头继续强势
  看空声音存在   → 看空消失  → 板块成交放大
```

## EMO 情绪周期定位（V26 新增）

### 四阶段

| 阶段 | 连板数 | 涨停板 | 情绪指数 | 操作 |
|------|--------|--------|---------|------|
| 启动期 | 1-2 | ≤5 | 30-45 | 🟢开始介入 |
| 发酵期 | 2-4 | 5-15 | 45-65 | 🟢重仓 |
| 高潮期 | 5+ | 15+ | 65-85 | 🔴逐步兑现 |
| 退潮期 | 0-1 | ≤3 | >85→30 | 🔴空仓 |

### 买在分歧/卖在一致

```
买在分歧：启动→发酵拐点，分歧5维中3维转一致 → 介入
卖在一致：高潮→退潮拐点，一致5维中3维转分歧 → 兑现
```
