# event-intelligence — 事件情报流派 V28

> **版本**：V28  
> **来源**：合并 `event-sentiment` + `news-signal`  
> **核心理念**：新闻是市场情绪的催化剂。事件情报远比"材料短缺"宽——包括政策、业绩、产品、供应链、宏观、评级、回购/减持等全部类型。  
> **五种信号模式**：黄金坑 / 板块早信号 / 政策催化 / 业绩驱动 / 资金事件

---

## 数据采集（全部必须）

```bash
# 1. 标的新闻（最近20条）
westock-data news <code> --limit 20 --type 2

# 2. 公告（重要事件）
westock-data notice <code> --limit 10

# 3. 业绩预告（A股）
westock-data reserve <code>

# 4. 大盘/宏观日历
westock-data calendar --limit 10 --country 1 --indicator 2

# 5. 市场资讯（重要宏观）
westock-data marketnews hs --limit 10

# 6. 板块排行（看板块催化剂）
westock-data sector --rank interval_chg_rank_sw1 --sort chg5Days --limit 20
```

## 信号模式一：黄金坑（高赔率）

### 判定
```
条件：3日跌幅 8-15% AND 此前横盘 ≥ 20日 AND 止跌确认

止跌确认（满足任一）：
  - 长下影线（下影线 > 实体 2倍）
  - 锤子线（收盘 > 开盘，且下影线 > 实体）
  - 次日阳线反包

新闻情绪（决定是否是真坑）：
  - 靴子落地（利空出尽）→ +3分
  - 恐慌蔓延（未知利空）→ -2分
  - 业绩暴雷 → 直接跳过，不是黄金坑
  - 无相关新闻 → +1分
```

### 评分
```
黄金坑分 = 跌幅分×0.3 + 止跌分×0.3 + 新闻情绪分×0.4

| 黄金坑分 | 信号 |
|---------|------|
| 4.0-5.0 | 🟢🟢 强黄金坑，重点关注 |
| 3.0-3.9 | 🟢 黄金坑，关注 |
| 2.0-2.9 | ⚠️ 疑似，不急 |
| <2.0 | ❌ 不是黄金坑 |
```

## 信号模式二：板块早信号（板块行情初期）

### 判定
```
催化类型 × 0.4 + 时机 × 0.3 + 强度 × 0.3

催化类型分：
  顶层政策（国务院/中央政治局）→ 5分
  部委政策（工信部/发改委/证监会）→ 4分
  行业会议/协会文件 → 3分
  供应链事件（涨价/限售/扩产）→ 3分
  大股东/机构增持 → 2分
  研报首次覆盖/大幅上调 → 2分
  无明显催化 → 1分

时机分：
  5日涨幅 3-8%（刚启动）→ 5分
  5日涨幅 8-15%（中段）→ 3分
  5日涨幅 >15%（末端）→ 1分

强度分：
  三渠道确认（新闻+公告+资金）→ 5分
  两渠道确认 → 3分
  单渠道 → 2分
```

### 评分
```
| 早信号分 | 信号 |
|---------|------|
| 4.0-5.0 | 🔥 早信号确认，可进观察 |
| 3.0-3.9 | 🟡 有催化，关注持续性 |
| 2.0-2.9 | ⚠️ 弱信号 |
| <2.0 | ❌ 无信号 |
```

## 信号模式三：政策催化（A股最重要）

### 判定
```
政策等级：
  顶层定调（中央/国务院）→ A级
  部委重要文件 → B级
  地方/协会文件 → C级

政策方向：
  直接利好 → +2分
  间接利好 → +1分
  中性 → 0分
  利空 → -2分

与标的相关性：
  主营业务直接受益 → 完整采纳
  沾边/概念 → 折半采纳
  无关 → 不采纳
```

## 信号模式四：业绩驱动

### 判定
```
业绩预告/快报：
  净利润超预期20%+ → +3分（强利好）
  净利润超预期5-20% → +2分（利好）
  符合预期 → +1分
  低于预期 → -2分
  亏损扩大 → -3分（强利空）

营收增速：
  营收增速 > 30% → +1分
  营收增速 > 50% → +2分
  营收负增长 → -1分

注意：
  扣非净利润 vs 净利润差距大 → -1分（盈利质量差）
```

## 信号模式五：资金事件

### 判定
```
大股东/实控人增持：
  大比例（>1%）增持 → +3分
  公告增持计划 → +2分
  小比例增持 → +1分

大股东/实控人减持：
  大比例减持 → -3分
  小比例减持 → -1分

回购：
  大比例回购（>2%）→ +2分
  小比例回购 → +1分
  激励回购（无实质）→ 0分

股权激励/定增：
  正常 → 0分
  定价远低于现价（利益输送嫌疑）→ -1分
```

## 综合事件评分

```
综合分 = Σ(各信号得分) / 5（归一化到5分制）

| 综合分 | 评级 | 判定 |
|:------:|:----:|:----:|
| 4.0-5.0 | ★★★★★ | 强催化剂，优先关注 |
| 3.0-3.9 | ★★★★ | 中等利好，积极关注 |
| 2.0-2.9 | ★★★ | 中性，看其他维度 |
| 1.0-1.9 | ★★ | 中等利空，谨慎 |
| <1.0 | ★ | 强利空，回避 |
```

## 数据完整性约束

```
news MISSING → 事件评分上限3分
calendar MISSING → 宏观政策维度缺失，评分上限3分
reserve NOT_APPLICABLE → 港股跳过业绩维度
marketnews MISSING → 宏观维度缺失，权重重分配
```
