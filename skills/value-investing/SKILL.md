## 分析流程（五步法）

### 第一步：巴菲特 + 段永平「好公司」定性评估

#### 1.1 公司业务（Business）
- [ ] 公司主要卖什么产品/服务（1-2句话说清）
- [ ] 核心客户是谁（ToC/ToB，地域结构）
- [ ] 行业位置（龙头/追赶者/细分强者）
- [ ] 增长驱动来自哪里（量、价、新市场、并购）

#### 1.2 盈利模式（Business Model）
- [ ] 收入结构（一次性/订阅/佣金/广告/金融）
- [ ] 利润结构（毛利率、经营杠杆、净利率）
- [ ] 现金流来源（OCF是否稳定，FCF是否可持续）
- [ ] 资本开支属性（维持性Capex vs 增长性Capex）

#### 1.3 公司文化与管理层
- [ ] 管理层是否诚实、透明、长期主义（巴菲特标准）
- [ ] 资本配置能力（回购/分红/并购/负债纪律）
- [ ] 是否符合"能不能睡得着觉"标准（段永平）

#### 1.4 巴菲特四问
1. 业务是否简单可理解（能力圈）
2. 是否有可持续护城河
3. 管理层是否可信且会配置资本
4. 当前价格是否有安全边际

---

### 第二步：段永平五维护城河评分

| 护城河类型 | 评估指标 | 1-5分 | 权重 |
|-----------|---------|-------|------|
| **网络效应** | 用户规模、平台双边市场、生态粘性 | 评分 | 20% |
| **品牌护城河** | 品牌认知度、定价权、用户忠诚度 | 评分 | 20% |
| **技术护城河** | 专利壁垒、R&D投入、技术领先性 | 评分 | 20% |
| **成本优势** | 规模经济、运营效率、单位成本 | 评分 | 20% |
| **切换成本** | 数据沉淀、服务依赖、迁移难度 | 评分 | 20% |

**综合评级**：
- 4.5-5.0: 极宽护城河 ⭐⭐⭐⭐⭐
- 4.0-4.4: 宽阔护城河 ⭐⭐⭐⭐
- 3.0-3.9: 中等护城河 ⭐⭐⭐
- <3.0: 护城河狭窄 ⭐⭐

---

### 第三步：现金流深度分析

```
经营现金流 (OCF) = 净利润 + 折旧摊销 + 营运资本变动
自由现金流 (FCF) = OCF - 资本支出 (CapEx)
每股FCF = FCF / 总股本
```

**现金流质量评估**：

| 指标 | 健康标准 | 分析意义 |
|------|---------|---------|
| FCF/净利润 | >70% | 盈利含金量 |
| OCF/营收 | >15% | 营收变现能力 |
| CapEx/OCF | <50% | 再投资强度 |
| P/FCF | <20x | FCF估值水平 |
| FCF Yield | >5% | 现金流回报率 |

---

### 第四步：多维度估值

#### 4.1 Forward P/E 估值

| 评级 | 标准 |
|------|------|
| 显著低估 | <行业均值-30% |
| 低估 | <行业均值-15% |
| 合理 | 行业均值±15% |
| 高估 | >行业均值+15% |
| 严重高估 | >行业均值+30% |

#### 4.2 PEG 估值

| PEG | 评级 |
|-----|------|
| <0.5 | 显著低估 |
| 0.5-1.0 | 合理偏低 |
| 1.0-1.5 | 合理 |
| 1.5-2.0 | 偏高 |
| >2.0 | 高估 |

#### 4.3 三阶段 DCF 模型

```
阶段1：高增长期 (3年) — 增长率 15-25%
阶段2：中等增长期 (2年) — 增长率 8-12%
阶段3：永续增长期 — 增长率 2-3%
WACC: 8-12% (根据行业调整)
```

**敏感性分析**：9种情景（增长率±5% × WACC±2%）

#### 4.4 安全边际判断

| 安全边际 | 评级 | 操作 |
|---------|------|------|
| >30% | ✅ 充足 | 可重仓 |
| 15-30% | ✅ 适中 | 分批建仓 |
| 5-15% | ⚠️ 偏小 | 小仓位 |
| <5% | ❌ 不足 | 等待 |

---

### 第五步：左侧交易计划

```
目标仓位: 5-10%（价值投资通常不重仓单只）

分批建仓（金字塔法）：
• 第一批 (试仓20%): 当前价±5%
• 第二批 (加仓30%): 回调10-15%
• 重仓区 (35%): 回调20-25%
• 极端区 (15%): 回调>30%（恐慌买入）

"时间是朋友"持有逻辑：
1. 护城河保护企业长期竞争力
2. ROE 显示资本回报能力
3. 自由现金流创造价值
4. 安全边际充足
5. 持有3-5年，让企业价值复合增长
```

## 与其他学派的协作

| 场景 | VALUE输出 | 七学派输出 | 协作方式 |
|------|----------|-----------|---------|
| 好公司+安全边际充足 | 🟢买入 | 可能因趋势弱而观望 | VALUE提供长线信心 |
| 好公司+估值偏高 | 🟡等回调 | 趋势强时买入 | 以七学派为主 |
| 差公司+趋势强 | 🔴回避 | 可能因动量强而买入 | VALUE否决投机 |
| 差公司+估值低 | 🟡价值陷阱 | 均值回归学派可能买入 | 需交叉验证 |
