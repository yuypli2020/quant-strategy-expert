# genesis-regime — 市场区间检测 V27

> **版本**：V27  
> **定位**：所有分析决策的基准层。分析任何标的之前必须先判断区间。  
> **触发**：任何 P1 分析前（隐式）| 用户问「现在大盘怎么样」「市场区间」  
> **数据源**：`westock-data` 必须可用

---

## 核心设计理念

```
传统做法：先分析股票，再考虑大盘环境
V27 做法：先判断市场区间，再带着区间背景分析股票

区间背景决定：
- 股票分析的权重配置（基本面 vs 技术面）
- 仓位上限（M因子已经在做，但区间判断更丰富）
- 策略选择（趋势跟踪 vs 均值回归 vs 困境反转）
- 多情景分析的基准情景
```

westock-data news --limit 5 --type 2  # 最新重要宏观新闻

## 第二步：五维区间指标计算

### 指标1：趋势维度（权重30%）
```
MA排列：
  MA5 > MA20 > MA50 > MA200 → 强势多头（+2）
  MA5 > MA20 > MA50         → 偏多（+1）
  混合/纠缠                  → 中性（0）
  MA5 < MA20 < MA50         → 偏空（-1）
  MA200 < MA50 < MA20 < MA5 → 弱势空头（-2）

MACD状态：
  MACD柱扩大且 > 0 → 多头（+1）
  MACD柱缩小且 > 0 → 多头减弱（+0.5）
  MACD柱扩大且 < 0 → 空头（-1）
  MACD穿越0轴     → 关键转折信号

趋势维度 = (MA得分 + MACD得分) / 2 → 归一化到 ±2
```

### 指标2：动量维度（权重25%）
```
计算：
  20日涨幅 > 8%  → 强动量（+2）
  20日涨幅 3-8%  → 中动量（+1）
  20日涨幅 -3~3% → 无动量（0）
  20日涨幅 -3~-8% → 负动量（-1）
  20日涨幅 < -8%  → 强负动量（-2）

注意：动量强 ≠ 上涨持续。动量极值往往是反转信号。
动量维度 = 原始得分（保留正负方向）
```

### 指标3：广度维度（权重20%）
```
涨跌家数（需要涨停数据或涨跌比）：
  上涨 > 65% → 广度健康（+1）
  上涨 55-65% → 广度一般（+0.5）
  上涨 45-55% → 背离警告（-0.5）
  上涨 < 45% → 广度恶化（-1）

新高新低（如果有数据）：
  新高股 > 新低股 → 强势（+1）
  新高股 < 新低股 → 弱势（-1）
  否则 → 中性（0）

广度维度 = 得分归一化到 ±1
```

### 指标4：防御维度（权重15%）
```
防御板块表现：
  银行/食品饮料/医药 涨幅 > 科技 → 资金防御（+1，谨慎信号）
  科技/消费/成长 涨幅领先 → 资金进攻（-1，积极信号）

国债/黄金表现：
  国债上涨 + 黄金上涨 → 避险情绪（+1，谨慎）
  国债/黄金下跌       → 风险偏好（-1，积极）

防御维度 = 综合得分归一化到 ±1
```

### 指标5：外部风险维度（权重10%）
```
汇率风险（如果涉及港股/美股）：
  USD/CNY > 7.3 → 高风险（+1）
  USD/CNY > 7.5 → 极高风险（+2）

地缘政治（从 news 判断）：
  有重大外部冲击 → +1~+2（按严重程度）

无重大外部风险 → 0
外部风险维度 = 风险溢价归一化
```

## 第三步：综合区间评分

```
区间总分 = 趋势×0.30 + 动量×0.25 + 广度×0.20 + 防御×0.15 + 外部×0.10

区间判定：
  3.0 ~ 5.0  → 🔴 **强势区间（Phase 1-2）**
  1.5 ~ 3.0  → 🟡 **偏强区间（Phase 3）**
 -1.5 ~ 1.5 → ⚪ **中性震荡（Phase 4）**
 -3.0 ~ -1.5 → 🟠 **偏弱区间（Phase 5）**
 -5.0 ~ -3.0 → 🔴 **弱势区间（Phase 6）**
```

## 第四步：区间相位解读

| 相位 | 名称 | 特征 | 最佳策略 | 仓位上限 |
|------|------|------|---------|---------|
| **Phase 1** | 牛市初期 | 均线多头 + 缩量调整后放量 | 均值回归/价值发现 | 80-100% |
| **Phase 2** | 牛市主升 | 趋势强劲 + 动量极值 + 成长领涨 | 趋势跟踪/动量 | 80-100% |
| **Phase 3** | 牛市末期 | 防御板块补涨 + 广度背离 | 减仓/价值陷阱检查 | 50-60% |
| **Phase 4** | 震荡市 | MA纠缠 + 动量中性 | 高抛低吸/事件驱动 | 30-50% |
| **Phase 5** | 熊市初期 | 动量转负 + 广度恶化 | 轻仓/困境反转/做空 | 10-30% |
| **Phase 6** | 熊市末期 | 趋势极弱 + 价值股抗跌 | 黄金坑/定投 | 10-30% |

## 第五步：区间驱动力识别（解释"为什么"）

```
从数据中识别当前区间的主要驱动力：

政策驱动型：
  - 降准/降息/财政刺激
  - 监管政策变化（科创板/注册制）
  - 外资准入放开
  → 特征：政策文件驱动，板块轮动快
  → 注意：政策顶往往领先经济底

盈利驱动型：
  - 企业财报EPS超预期
  - 行业景气度提升
  → 特征：业绩公告驱动，成长股先行
  → 注意：盈利顶往往滞后价格顶

资金驱动型：
  - 北向资金持续流入
  - 融资余额增加
  - 居民储蓄搬家
  → 特征：流动性驱动，估值扩张
  → 注意：资金顶往往领先盈利顶

外部驱动型：
  - 美联储政策/关税/地缘政治
  → 特征：突发性强，不可预测
  → 注意：外部冲击往往短暂，A股有政策对冲

不明型（最难处理）：
  - 各指标矛盾，无明显驱动力
  → 建议：降低置信度，等明确信号
```

## 第六步：区间预测（前瞻性判断）

```
未来1-3个月区间展望：
基于当前相位 + 驱动力 + 催化剂日历

催化剂日历：
  - 财报季（3月/4月/8月/9月）：偏多倾向
  - 美联储FOMC会议：外部风险窗口
  - 重要政策会议（两会/中央经济工作会议）：政策催化剂
  - 解禁/减持高峰：供给压力

驱动力持续性：
  - 政策驱动：持续性中等，政策顶后减弱
  - 盈利驱动：持续性好，跟随盈利周期
  - 资金驱动：持续性较差，跟随情绪
  - 外部驱动：持续性弱，难以预测
```

## 与其他模块的集成

```
genesis-regime 作为基础层，其他模块调用：

genesis-scan（扫描）：
  → Step 0 先调用区间检测 → 调整仓位预期
  → 区间为Phase 5-6时 → 黄金坑扫描优先
  → 区间为Phase 1-2时 → 板块早信号优先

genesis-v22（分析）：
  → Step 0 先加载区间 → 确定分析背景
  → 多情景分析的基准情景 = 当前相位
  → 仓位上限 = min(regime仓位, M因子仓位)

portfolio-constructor（组合）：
  → 不同区间用不同仓位配置
  → Phase 1-2：成长股+动量股权重高
  → Phase 5-6：防御股+黄金坑权重高
  → Phase 4：低相关多空对冲

genesis-evolve（迭代）：
  → 按区间统计策略胜率
  → "趋势策略在Phase 2胜率80%，Phase 4胜率仅30%"
  → 调整策略权重配置
```
