## 反复踩过的坑

| 坑 | 教训 | 修复 |
|-----|------|------|
| 华润微误归类为"硅片股" | 板块标签≠业务本质。必须用 `profile` 查主营业务再判断产业链位置 | 第十一条：每次板块推荐必须验证产业链位置 |
| warmup 切片忘了传 start/end | 导致 Sharpe/最大回撤被 warmup 段 NaN 污染 | 第三条：warmup 后必须重算评估指标 |
| MACD金叉用 `&` 写条件 | westock-tool 多条件必须用 `intersect([])`，不支持 `&` | 详见 TOOLS.md filter 表达式语法 |
| A股默认仅多头没问用户 | 做空 A股必须先问清方案 | 第二条：A股做空必须先问 |
| 隐式 look-ahead 没警告 | "今天涨X%买"类描述必须先警告 | 第一条：look-ahead 必须先警告 |
| PE 负值没排除 | 亏损股 PE 为负，必须先 `PE_TTM > 0` | 详见 TOOLS.md filter 表达式语法 |

## 数据源限制备忘

> 以下限制仅在各 skill 可用时适用。skill 不可用时，相关限制自然不生效，但替代数据源可能有额外限制，须在交付时说明。

| 数据源 | 限制 | 替代方案 |
|--------|------|---------|
| `westock-data` | 龙虎榜/大宗/两融仅沪深；筹码仅沪深京A；股东仅A股/港股；港美货币必须保持 HKD/USD | web_fetch / online-search，精度下降须告知 |
| `westock-tool` | CSI 300/CSI 500 不是全市场代理，有选样偏差 | 直接用指数或全市场筛选 |
| `neodata-financial-search` | 语义搜索结果不能进回测循环；公募基金不覆盖香港 | web_search 替代 |
| `quant-backtest-lab` | 不可用时无法完成标准回测流程 | **无降级，必须告知用户** |
